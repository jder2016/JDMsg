* 部门和工作地点未知？

 **1.关于部门和工作地点，我只问了商城研发的，商城研发在亦庄和北辰都有，具体的岗位目前还不知道，入职之后会分，到时候才能知道具体在哪工作
 2.个人被分配到哪个部门，这个和二面面试官有关，你现在打电话过去问hr能知道大部门，但是具体的小岗位还是不知道，一般就是二面面试官门下
 3.京东商城各部门的工作地点，这个跟部门有关，一般数字营销在北辰比较多，研发什么在亦庄比较多**
